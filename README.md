# KRONOS SYSTEM - Gestão Financeira

Sistema executivo de gestão financeira empresarial.

## 🚀 Funcionalidades

- Cadastro de clientes com controle de parcelas
- Separação por dia de vencimento (5 e 20)
- Filtro por mês de referência
- Status: Pago, Quitado, Finalizando, Inadimplente
- Dashboard executivo com contadores
- Integração com Firebase Realtime Database

## 📦 Instalação

1. Clone o repositório
2. Configure o Firebase em `js/firebase-config.js`
3. Abra `index.html` no navegador

## 🔧 Configuração Firebase

1. Crie um projeto no [Firebase Console](https://console.firebase.google.com/)
2. Ative o Realtime Database
3. Copie as credenciais para `firebase-config.js`

## 🛠️ Tecnologias

- HTML5, CSS3, JavaScript
- Bootstrap 5
- Firebase Realtime Database
- Font Awesome

Compatível com desktop e dispositivos móveis.
